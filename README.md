# Hand Gesture Controlled Car

## Description
Developed a wireless gesture-controlled robotic car using accelerometer (MPU6050) and Arduino. The system processes real-time sensor data to control movement based on hand gestures.

## Features
- Real-time gesture recognition using accelerometer data
- Wireless control using Bluetooth communication
- Mapping gestures to directional movements (forward, backward, left, right)
- Integrated obstacle detection for safety

## Technologies Used
- Arduino (C/C++)
- Embedded C
- I2C Communication (MPU6050)
- Bluetooth Module (HC-05)
- Sensor Data Processing

## Key Concepts
- Real-time data processing
- Gesture recognition algorithms
- Wireless communication protocols
- Embedded system programming
